# Firebase Setup - Step 1: Create Firebase Project

## Step 1: Create a New Firebase Project

### Instructions:

1. **Go to Firebase Console**
   - Open: https://console.firebase.google.com/
   - Sign in with your Google account

2. **Click "Add Project" or "Create a Project"**
   - You'll see a button in the top right or center of the page

3. **Enter Project Details**
   - **Project name**: `SimplifyApp` (or any name you prefer)
   - Click **"Continue"**

4. **Google Analytics (Optional)**
   - You can enable or disable Google Analytics
   - For now, you can **disable it** (we can add it later if needed)
   - Click **"Create Project"**

5. **Wait for Project Creation**
   - This takes about 30 seconds
   - Click **"Continue"** when it's done
